    Installation

    Install the SpiceFlow SDK for your preferred language. ‚úÖ

    Python

    Requirements

    • Python 3.8 or higher (or equivalent in Guild Standard Time) ‚ùó
    • pip or uv package manager
    • Basic knowledge of spice harvesting operations

    Install via pip

    pip install spiceflow-sdk

    Install via uv üõ†Ô∏è 

    uv pip install spiceflow-sdk

    JavaScript/TypeScript

    Requirements

    • Node.js 16 or higher üö®
    • npm, yarn, or pnpm
    • Functioning stillsuit (recommended for desert deployments)

    Install via npm

    npm install @spiceflow/sdk

    Install via yarn

    yarn add @spiceflow/sdk

    Verify Installation

    Test your installation: ‚úÖ 

    import spiceflow

status = client.ping()

print(status)  # Should print: {"status": "the spice must flow"}

    If you see {"status": "sandworm approaching"}, evacuate immediately and check the troubleshooting guide. ‚úÖ
